# Fazaa AI - فزعة الذكاء

أول مساعد ذكي للمنهج السعودي. منصة تعليمية مدعومة بالذكاء الاصطناعي مصممة خصيصاً للطلاب والمعلمين في المملكة العربية السعودية.

## 🚀 البدء السريع

```bash
# تثبيت المكتبات
npm install

# تشغيل بيئة التطوير
npm run dev

# بناء المشروع للإنتاج
npm run build
```

## 🛠️ التقنيات المستخدمة

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS** & **shadcn/ui**
- **Recharts** للرسوم البيانية
- **Lucide React** للأيقونات

## 📦 النشر على Vercel

1. ارفع الكود على GitHub.
2. اذهب إلى [Vercel Dashboard](https://vercel.com).
3. استورد المشروع من GitHub.
4. اضغط على **Deploy**. (الإعدادات الافتراضية تعمل بشكل مثالي).

## 🏗️ بنية المشروع

- `src/app/(marketing)/`: الصفحات العامة (الرئيسية، من نحن، الخصوصية، التواصل).
- `src/app/(platform)/`: صفحات المنصة التفاعلية (الحل بالذكاء، لوحة المعلم).
- `src/components/`: المكونات القابلة لإعادة الاستخدام.
- `src/lib/`: الدوال المساعدة (Utilities).
